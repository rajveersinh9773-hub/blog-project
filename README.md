# MyBlog - React Blog Website

A modern and responsive blog website built using React.js, Tailwind CSS, Axios, and JSON Server.

## Features

- Responsive blog website design
- Featured blog section 
- Fetch blog data using Axios
- JSON Server API integration
- Category section
- Newsletter subscription form
- Modern footer design
- Hover animations

## Technologies Used

- React.js
- Tailwind CSS
- React Icons
- Axios
- JSON Server
- JavaScript

## Installation

Install dependencies:

```bash
npm install
Run React App

Start the development server:

npm run dev

App runs on:

http://localhost:5173
Run JSON Server

Install JSON Server:

npm install json-server

Start server:

json-server --watch db.json

API URL:

http://localhost:3000/blogs
